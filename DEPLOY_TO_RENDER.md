## 📋 To Deploy on Render

### 1️⃣ Push to GitHub

```bash
cd D:\MERN_LEARN\job-scraper-platform
git init
git add .
git commit -m "Job Scraper Platform Full Stack"
git remote add origin https://github.com/YOUR_USERNAME/job-scraper-platform.git
git push -u origin main
```

### 2️⃣ Go to Render.com

1. Sign up or login: https://render.com
2. Click **"New"** → **"Web Service"**
3. Select **"Deploy existing Git repository"**
4. Find `job-scraper-platform` repo and connect

### 3️⃣ Configure on Render

**Build Command:**
```
npm install && cd client && npm install
```

**Start Command:**
```
npm run build && cd server && npm start
```

**Environment Variables** (Add all):
```
MONGO_URI=mongodb+srv://username:password@cluster.mongodb.net/job-scraper
JWT_SECRET=your_secret_key_32_chars_minimum
OPENAI_API_KEY=sk-... (optional)
NODE_ENV=production
PORT=5000
```

### 4️⃣ Deploy

- Click **"Create Web Service"**
- Wait 5-15 minutes for build
- Get your live URL
- Done! ✅
